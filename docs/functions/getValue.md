# Function: getValue()

> **getValue**(`namespace`): `string` \| `null`

Returns the value of a variable in the datalayer

## Parameters

### namespace

`string`

Namespace of the variable (can be separated with a dot)

## Returns

`string` \| `null`

Value of a variable or null if it does not exist
