# Function: setValue()

> **setValue**(`namespace`, `value`): `void`

Creates the required namespace in a data layer object
and overwrites it if it already exists

## Parameters

### namespace

`string`

Namespace of the variable (can be separated with a dot)

### value

`string`

Value of the variable

## Returns

`void`
